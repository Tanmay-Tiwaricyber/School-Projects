Python & MySQL Database Creation


import mysql.connector

# Establish connection to MySQL
mydb = mysql.connector.connect(
    host="localhost",
    user="root",
    password="root"
)

mycursor = mydb.cursor()

# Create database if it doesn't already exist
mycursor.execute("CREATE DATABASE IF NOT EXISTS JNVM_SCHOOL")
print("Database 'JNVM_SCHOOL' created successfully!")

Explanation:

  1. Import mysql.connector: This module allows Python to interact with MySQL databases.
  2. Connect to MySQL: Use mysql.connector.connect to establish a connection to the MySQL server.
  3. Create Cursor: A cursor is used to execute SQL queries.
  4. Execute Query: The SQL query CREATE DATABASE IF NOT EXISTS JNVM_SCHOOL creates the database if it doesn't already exist.
  5. Print Confirmation: Displays a success message once the database is created.

Example:


# Running the script will create the 'JNVM_SCHOOL' database
Database 'JNVM_SCHOOL' created successfully!

Note:

Ensure MySQL is running on your system, and replace host, user, and password with your MySQL server credentials.
