MySQL Drop Table Query

import mysql.connector

# Establish connection to MySQL
mydb = mysql.connector.connect(host="localhost", user="root", password="root",database="school")

mycursor = mydb.cursor()

# Execute the DROP TABLE query
mycursor.execute("DROP TABLE students")

# Fetch and print all records
for x in mycursor:
    print(x)

Explanation:

  1. Import the mysql.connector module to interact with the MySQL server.
  2. Establish a connection to the MySQL server using mysql.connector.connect(), providing the host, username, and password.
  3. Create a cursor object to execute SQL queries.
  4. Execute the DROP TABLE query using mycursor.execute("DROP TABLE students") to remove the students table from the database.
  5. Use a for loop to attempt to fetch and print any records returned by the query, though in the case of DROP TABLE, there are no records returned.

Key Points:

  • DROP TABLE students: Deletes the students table from the database.
  • The DROP TABLE command is irreversible; once the table is dropped, all data and structure are lost.
  • No results are returned after dropping the table, so the for loop won't print anything in this case.
  • The cursor object allows you to interact with the database and execute queries.
