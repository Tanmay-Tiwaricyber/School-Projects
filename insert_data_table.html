MySQL Insert Query

import mysql.connector

# Establish connection to MySQL
mydb = mysql.connector.connect(host="localhost", user="root", password="root", database="school")

mycursor = mydb.cursor()

sql = "INSERT into students(name, age) VALUES (%s, %s)"
val = ("Abhay", 20)

mycursor.execute(sql, val)
mydb.commit()

print("Data Inserted")

Explanation:

  1. Import the mysql.connector module to interact with the MySQL server.
  2. Establish a connection using mysql.connector.connect() by providing the host, username, and password.
  3. Create a cursor object to execute SQL queries.
  4. Define the SQL INSERT query: INSERT INTO students(name, age) VALUES (%s, %s).
  5. Provide the values to insert into the table, e.g., ("Abhay", 20).
  6. Execute the query using mycursor.execute() and pass the query and values as parameters.
  7. Commit the changes using mydb.commit() to save the data into the database.
  8. Print a success message to confirm the insertion.

Key Points:

  • mydb.commit(): Ensures that the transaction is saved to the database.
  • Use parameterized queries (%s) to prevent SQL injection.
