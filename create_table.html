MySQL Create Table Notes

import mysql.connector

# Establish connection to MySQL
mydb = mysql.connector.connect(host="localhost", user="root", password="root")

mycursor = mydb.cursor()

mycursor.execute("CREATE TABLE Students (
    Name VARCHAR(100) NOT NULL,
    Age INT NOT NULL
)")

for x in mycursor:
    print(x)

Explanation:

  1. Connect to MySQL: Establishes a connection to the MySQL server using the provided host, username, and password.
  2. Create Cursor: A cursor is created to execute SQL commands.
  3. CREATE TABLE: This command creates a new table named Students with the following columns:
    • Name: A string (up to 100 characters) that cannot be NULL.
    • Age: An integer to store the studentâ€™s age, also cannot be NULL.
  4. Iterate Cursor: Loops through the cursor to print the output of the query execution.
